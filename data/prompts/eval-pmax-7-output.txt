## Outcome digest

**What it does**
Reviews a diff and emits a Markdown review comment listing every block whose logic duplicates code already present elsewhere in the codebase, with severity and cross-reference for each finding.

**Key inputs**
The current diff against the default branch and the existing src/**/*.py and src/**/*.ts files.

**Done when**
Every duplicated block in the diff has a finding with severity, file path, line range, and a cross-reference to the other location.

**Quick sample**

    - **high** src/checkout/total.py L42-L58 duplicates src/cart/summary.py L17-L31 (both sum line items with the same tax rule)
    - **medium** src/api/format.ts L88-L102 duplicates src/reports/format.ts L12-L26 (identical date-formatting branch)
    - **low** src/utils/retry.py L4-L11 duplicates literal backoff sequence from src/utils/net.py L55
