# @jl-cmd/prompt-generator

Standalone prompt-generator and agent-prompt skills for Claude Code, extracted from [claude-code-config](https://github.com/jl-cmd/claude-code-config).

## Install

```bash
npx @jl-cmd/prompt-generator
```

This copies skills, hooks, and rules into `~/.claude/`.

## Contents

- **skills/prompt-generator/** — Repository-grounded XML prompt authoring
- **skills/agent-prompt/** — Structured agent prompt crafting
- **hooks/blocking/** — Prompt workflow gate and validation hooks
- **rules/** — Prompt workflow context controls
